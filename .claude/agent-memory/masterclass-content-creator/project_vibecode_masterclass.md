Masterclass created for the "Vibecode with Safety" project on 2026-04-04. Co-hosted format: HOST (Jorge, tech instructor) and LAWYER (guest from Sonae Sierra, legal expert). Covers 9 topics across 3 hours with a 10-minute break in the middle. Part 1 is technical foundations (LLMs, prompts, AGENTS.md, SKILL.md, ARCHITECTURE.md). Part 2 is safety/law/practice (SPEC.md, subagents, GDPR/PII, vibecoding workflow). Target audience is intermediate Python backend developers at ETIC Algarve.

**Why:** Jorge wants to teach AI-assisted development with a strong safety emphasis -- particularly GDPR and PII awareness -- so students build safe habits from day one. The co-hosted format with a real lawyer adds credibility and legal grounding.

**How to apply:** Future content for this project should maintain the safety-first framing. GDPR/PII is the centerpiece, not an afterthought. Technical topics should always connect back to responsible usage. The co-hosted dynamic (HOST/LAWYER handoffs) is a key pedagogical choice.
