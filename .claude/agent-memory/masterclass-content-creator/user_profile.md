Jorge Alves teaches Backend II at ETIC Algarve (Portugal). His students are intermediate Python developers familiar with web frameworks, APIs, and basic deployment. He teaches in a practical, safety-conscious style that blends technical depth with real-world awareness (GDPR, PII, secure development practices). For the "Vibecode with Safety" masterclass, he uses a co-hosted format with a legal expert from Sonae Sierra as guest (LAWYER role handles GDPR/PII/legal topics, HOST handles technical topics). Project repo lives under `~/projects/etic_algarve/`.
